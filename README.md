## 使用方法

### 客户端

访问[这里](https://impostor.github.io/Impostor/)下载文件以连接到服务器。

### 服务端

1. 安装[.NET 6.0 Runtime](https://dotnet.microsoft.com/zh-cn/download)，当然SDK也可以。
2. 下载最新版本。
3. 下载适用于服务端操作系统的版本，注意分清楚架构（x64或arm）。
4. 解压。
5. 更改config.json
6. 运行`Impostor.Server`或`Impostor.Server.exe`。

> 并不适用 Docker。
